## 🧐 왜 만들었나요?

OLED 모니터를 샀는데 **번인(Burn-in)**이 걱정되었습니다.
그래서 윈도우 기본 기능인 '작업 표시줄 자동 숨기기'를 한동안 사용했는데, 너무 힘들었습니다.

1.  일단 버그가 많습니다.
2.  창의 가장 아래쪽에 있는 버튼을 누르려고 하면, 작업 표시줄이 튀어 올라와서 클릭을 방해합니다.

그래서 하나 만들어봤습니다. 이 프로그램은 작업 표시줄을 아래로 숨기는 게 아니라, **투명도(Opacity)만 0으로** 만듭니다. 자리는 그대로 차지하고 있어서 OLED는 보호되지만, 갑자기 튀어나오면서 방해하지 않습니다.

> **참고:** AI가 써준 소개 글은 무슨 대단한 기업용 솔루션 같았는데, 사실 아닙니다. 그냥 직접 쓰려고 만든 아주 단순한 토이 프로젝트입니다. 누군가에게는 필요할지도 몰라서 올려봅니다. 심지어 영어부터 작성하고 한국어로 번역하니까 요상한 어투가 되었네요... 귀찮으니 그냥 씁니다.

## ✨ 기능

* **단순한 로직**: 평소에 작업 표시줄을 투명하게 만듭니다. 끝이에요.
* **나름 지능적인 가시성**: 시작 메뉴나 검색 창, 알림 센터가 뜨면 작업 표시줄이 다시 보입니다. (작업 표시줄 없이 시작 메뉴만 둥둥 떠 있으니까 너무 이상해서 넣었습니다.)
* **시간 조절 가능**: 숨겨지는 딜레이 등을 조절할 수 있습니다. 솔직히 이거 말고는 설정할 게 없어서 넣었습니다.
* **"가벼움"**: 소스 코드는 정말 가볍습니다. 메모리 사용량은... 음... .NET이잖아요? 작업 관리자는 흐린 눈으로 봐주세요. 🥲

## 📥 설치 방법

멋진 설치 파일을 만드는 법을 아직 모릅니다. 그냥 릴리즈에서 실행 파일을 받아주세요.

1.  [**Releases**](https://github.com/LI-NA/HideTB/releases) 페이지로 이동합니다.
2.  사전 빌드된 zip 파일을 다운로드합니다.
3.  압축을 풀고 `HideTB.exe`를 실행합니다.
